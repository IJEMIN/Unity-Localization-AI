# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-02-26

### Added
- Initial release of Unity Localization AI.
- Automated translation for Unity String Tables using OpenAI's GPT models.
- Support for GPT-4o, GPT-4o-mini, and other OpenAI models.
- Preview translation feature to test prompts before batch processing.
- Customizable system prompts for specialized translation rules.
- Preferences window for API key and model configuration.
- Editor window for managing and executing translation tasks.
